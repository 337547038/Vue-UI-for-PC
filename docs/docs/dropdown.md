## API
### BackTop
|参数|类型|说明|
|-|-|-|
|height         | Number/200     |页面滚动高度达到该值时才显示BackTop组件|
|bottom         | Number/30      |组件距离底部的距离|
|right          | Number/30      |组件距离右部的距离|
|text           | String         |显示文本|
|click          | Function       |点击事件|
